

## Usage

```bash
trmonitor
trmonitor --lang tr
trmonitor --lang en
```

| Option | Description | Default |
|--------|-------------|---------|
| `--lang` | Output language (`tr` or `en`) | Auto-detect from OS |
| `--log` | Show application logs | - |
| `--version` | Show version | - |
| `--help` | Show help | - |
